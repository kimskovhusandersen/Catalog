Project Catalog

Introduction

Create, read, update and delete Categories, Items and Users in the Catalog with this API.

Overview

This API is developed as part of the second project of the Full Stack Web Developer Nanodegree at Udacity.

Authentication

GET requests are publicly available. All other requests requires a free login.

Error Codes

If there's an error, a JSON object with the key [message] will be return. The value of the key will be the error message(s).

Rate limit

Currently, there's a rate limit of 240 requests per day on all GET requests.
